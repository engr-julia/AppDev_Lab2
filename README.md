# 🚀 PHP Array Challenges

Welcome to the PHP Array Challenges project!  
This mini-project is designed to strengthen foundational PHP skills through hands-on exercises focused on arrays and data manipulation.

Instead of just learning theory, this project applies real PHP functions to structured data, helping build confidence in working with arrays — one of the most important concepts in programming.

---

## 📚 Project Overview

This project combines three practical challenges into one learning journey.

It begins with working on a simple indexed array of numbers, where the program calculates the total sum and counts how many elements exist in the array. By using built-in PHP functions like `array_sum()` and `count()`, this part highlights how powerful and efficient PHP’s array utilities can be when performing mathematical operations.

The next section explores dynamic array manipulation using a list of colors. The array is sorted, expanded by adding new elements at both the beginning and end, and modified by replacing specific values. Functions such as `sort()`, `array_push()`, `array_unshift()`, and `array_search()` are used to demonstrate how flexible arrays can be. This challenge focuses on modifying and managing data efficiently.

The final section introduces a more advanced concept: a multi-dimensional associative array representing job listings. Each listing contains structured information including an ID, job title, company name, email, contact number, and a list of required skills stored in a nested array. New job records are added dynamically, and specific data is accessed through nested indexing. This challenge simulates real-world structured data handling and demonstrates how PHP manages complex data structures.

---

## 🧠 Skills Practiced

Throughout these challenges, the following concepts are applied:

- Creating and initializing indexed arrays
- Sorting and modifying array elements
- Using PHP built-in array functions
- Working with associative arrays
- Managing multi-dimensional arrays
- Accessing nested array data
- Structuring data in a clean and readable format
